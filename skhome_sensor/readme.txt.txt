Sensors:
- DHT21 : Low cost temperture and humidity sensor.
- DS18B20 : Water proof temperature sensor.
- SHT10 : Soil moisture with temperature sensor.

Required libraries:
- DHT21
- Adafruit_sensor
- Dallas Temperature
- OneWire
- SHTx
- Microgear

for OTA uploading, also install python 2.7 with set path. 

IDE: Arduino 1.8.2 + esp8266 board manager.
